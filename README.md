# typescript-react-sass-webpack-babel-boilerplate
This is a basic boilerplate for these technologies:
- TypeScript
- React
- SASS
- Webpack
- Babel

### Setup dev environment
1. Check this repo out.
2. `yarn install`
3. `yarn add core-js react react-dom regenerator-runtime`
4. `yarn add -D  @babel/core @babel/preset-env @babel/preset-react @babel/preset-typescript @types/react @types/react-dom babel-loader css-loader node-sass sass-loader source-map-loader style-loader webpack webpack-cli typescript ts-loader`
5. `yarn run webpack:dev`
6. open index.html
